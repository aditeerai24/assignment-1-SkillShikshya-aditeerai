Day-1 of Learning JavaScript

JavaScript (JS) is a versatile, high-level programming language primarily used for building interactive and dynamic websites.
JavaScript can be used to create desktop applications using frameworks like Electron.js.

Difference Between let and var

*let : Declares variables that are block-scoped, meaning they are only accessible within the block where they are defined.

*var: Declares variables that are function-scoped, meaning they are accessible throughout the function in which they are defined or globally if declared outside any function.
